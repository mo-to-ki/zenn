# zenn
## 使用方法
zenn cli のインストール
```
npm init
npx install zenn-cli
```

zenn cli のアップデート
```
npm install zenn-cli@latest
```

記事の作成
```
npz zenn new:article
```

プレビュー
```
npx zenn preview
```

> [!WARNING]
> ディレクトリからファイルを削除しても記事は消えません。  
> 削除は [ダッシュボード](https://zenn.dev/dashboard) から行ってください。
